
## 主要功能

多租户设计，实现容器云的多租户使用

基于k8s集群的容器云，集群管理，集群节点自动横向扩展，集群状态概览

容器管理，镜像管理，POD管理

应用发布，更新，应用服务的管理

容器云的监控告警
